🔒 Sicurezza dei dati
Tutti i dati rimangono esclusivamente sul dispositivo, nel localStorage del browser — nessun server, nessuna trasmissione automatica.

La condivisione avviene solo su tua iniziativa esplicita tramite il sistema di messaggistica del dispositivo (WhatsApp, SpikAd, ecc.).

Online è pubblicata unicamente l’applicazione vuota su GitHub Pages: nessun dato operativo lascia il circuito cellulare del gruppo di lavoro. Il codice sorgente è pubblicamente ispezionabile per verificare l’assenza di telemetria.

Files Tecnici — La sezione Condividi APP e Files Tecnici dà accesso a documentazione operativa sensibile: Sinno Stegni List, MDP NOGO SEE-IN e MDP NOGO SEE-OFF. Per ragioni di sicurezza operativa questi files non sono presenti sul server pubblico. Vengono distribuiti esclusivamente tramite condivisione peer-to-peer cifrata (WhatsApp) su richiesta esplicita. Una volta installati, rimangono memorizzati solo sul dispositivo locale e non vengono mai trasmessi in rete.

📲 Aggiungi alla Home Screen
🤖 Android — Chrome (metodo nativo)
1. Apri il file index.html in Chrome
2. Tocca il menu ⋮ (tre puntini in alto a destra)
3. Seleziona "Aggiungi a schermata Home"
4. Conferma il nome e tocca "Aggiungi"
⚠ Funziona solo se il file è aperto da un server locale o GitHub Pages, non da percorso file:///
🤖 Android — File locale (file:///)
Per creare un'icona che apra direttamente un file HTML locale usa Shortcut Maker:
1. Installa Shortcut Maker dal Play Store
2. Apri l'app → Activities
3. Cerca Chrome → seleziona com.android.chrome / Main
4. Imposta l'URL: file:///sdcard/Consumabili/index.html
5. Scegli un'icona → "Crea collegamento"
🔗 Shortcut Maker — Play Store
🍎 iOS — Safari (metodo nativo)
1. Apri il file in Safari
2. Tocca l'icona condividi (quadrato con freccia in su)
3. Scorri e seleziona "Aggiungi a schermata Home"
4. Conferma il nome → "Aggiungi"
Funziona da GitHub Pages. Per file locale usare l'app qui sotto.
🍎 iOS — File locale
Usa Launcher • Link Maker (ex Launcher Center Pro):
1. Installa l'app dall'App Store
2. Crea un nuovo launcher → tipo URL
3. Inserisci: shareddocuments:///index.html
4. Scegli nome e icona → aggiungi widget alla Home
In alternativa usa Scriptable per aprire file locali via script.
🔗 Launcher — App Store 🔗 Scriptable — App Store
✅ Soluzione universale — GitHub Pages
Se il file è già pubblicato su GitHub Pages, sia Android che iOS possono aggiungere l'icona alla Home nativamente (vedi passi sopra) senza app di terze parti.
L'app funziona offline dopo il primo caricamento grazie alla cache del browser.
